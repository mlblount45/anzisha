# anzisha


## Module Structure
The `name` field will be the alias used to start the module and will be used to add the module as a 
dependency to other modules.
###Json Representation
```
{
   "name":"moduleC",
   "scripts":[
      "command to run"
   ],
   "dependencies":[
      "moduleA",
      "moduleB"
   ]
}
```
###YAML Representation 
Not yet supported
